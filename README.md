# Getting Started

Make sure you have **Node.js** installed. You also need an **ITTYBIT_API_KEY** to run this project. You can get your API key from [https://ittybit.com/](https://ittybit.com/).  

Then, install the dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
```
Start the development Server
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
